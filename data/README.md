## Updating `data`

All data packages are generated by `tools/` from Minecraft server jar data.
To regenerate for a new MC version:

```bash
cd tools && go run . --extract --version X.YY.ZZ
```

This extracts JSON data from the MC server jar and runs all 8 generators,
producing updated `.go` files across `data/` and `level/`.

See [docs/dev/tools.md](../docs/dev/tools.md) for the full pipeline documentation.

### Packages

| Package | Generated From | Content |
|---------|---------------|---------|
| `packetid/` | `packets.json` | 262 packet ID constants |
| `soundid/` | `registries.json` | 1,838 sound ID constants |
| `item/` | `items.json` + `registries.json` | 1,505 item ID constants |
| `entity/` | `entities.json` (Java extractor) | 157 entity types with dimensions |
| `registryid/` | `registries.json` | 95 registries, 6,726 total entries |
| `lang/` | Mojang manifest (live) | Language translations (`go generate`) |
